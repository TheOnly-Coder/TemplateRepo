⚡ RepoWeb Lua API Demo

This page demonstrates all available Lua APIs in RepoWeb.

1. DOM Access

-- Get element and modify it
local el = __repoweb_dom.getElementById("target")
el.setHTML("Hello from Lua!")
el.setAttr("class", "highlight")

2. Lua Computation

-- Pure Lua (runs in Fengari VM)
function factorial(n)
  if n <= 1 then return 1 end
  return n * factorial(n - 1)
end
